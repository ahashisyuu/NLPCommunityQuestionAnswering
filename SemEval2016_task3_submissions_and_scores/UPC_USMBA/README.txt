5. System specs

  - 5.1 Core approach
	Supervised ML approach using Logistic Regression from Weka toolbox. 
	197 features used for ranking.
	Using Google Translator API for translating from Arabic into English all queries and pairs query/answers.
	Preprocess of both Arabic and English texts using Stanford CORE NLP.
	Enrichment of Arabic texts with Arabic WordNet.
	Enrichment of English texts using WordNet.
	Using a terminology (English & Arabic) of medical terms classified as Body Parts, Diseases, and Drugs
