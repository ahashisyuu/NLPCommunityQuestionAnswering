Deep Ranking of question-comments and question-question in cQA forums using Siamese Architecture.

Abstract: 
	Layered training of the Convolution based Siamese network is done to rank comments and related questions in cQA forums. First layer of training is done using 2 million data of question and their related answers available in Yahoo! dataset. This trained the model to discriminate between positive and negative answers. The second layer of training is done using Semeval 2016 dataset. This deeper layer trained the model to rank the related questions and comments provided for a particular question.
